Simple Answers to Questions which you may have

Can I modify my costumes in the newer Curation?

Yes! It’s possible using the setPart command in the Developer Console. IN-COMPLETE

How do I open the Developer Console?

By pressing the tilde (~) key. On most keyboards its shift + (the key beside 1 on the left)

Can I move my Savedata around onto another PC?

Yes. For this, these are it’s instructions:

  • Press Win + R and type %appdata%\Macromedia\Flash Player\#SharedObjects
  • Go into the Alphanumerical Junk named folder
  • Copy the ```www.poptropica.com`` folder
  • On the other device, run the curation and setup a new character
  • Close the curation
  • Go to the %appdata%\Macromedia\Flash Player\#SharedObjects folder and inside that alphanumrical folder.
  • Delete the www.poptropica.com folder and replace with the one you copied.
  • Done!

How do I delete my account?

  • Press Win + R and type %appdata%\Macromedia\Flash Player\#SharedObjects
  • Go into the Alphanumerical Junk named folder
  • Delete the www.poptropica.com folder

Help! The Poptropica Curation doesn’t seem to work on my Macbook

Macbook Support is very limited when it comes to Flashpoint. As well, the curation has never seemed to work well when we tried it on Macbooks.
