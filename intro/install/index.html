A Flashpoint curation containing both AS2 and AS3 islands

Installation

  • Firstly, Install 7Zip
  • Then Download and Unzip Flashpoint Core (You’re free to use Flashpoint Core 12, not tested though)
  • Download the AS3 Flashpoint Curation
  • Open up Flashpoint Core and go to the Curate tab and press Load Archive
  • Wait for the Curation to Load
  • Scroll Down and Press Run to start the Game!

NOTE: This is still being continously tested, so feel free to drop by for support or any bugs you encounter along the way on Poptropica Fan Club Discord Server
